Use Qwen to Run Agent

Build an agent that plans, calls tools, and iterates. Combine Qwen for reasoning with image/video generation to automate multi‑step tasks like asset creation, captioning, and publishing.

  • Define tools (generation, search, storage)
  • Let the agent plan and call tools with guardrails
  • Review results; the agent refines outputs automatically
# use qwen to run agent (pseudo)
goal: generar imagenes y video con qwen 2.5
tools: [image.generate, video.generate, search]
plan -> call tools -> review -> iterate -> publish
